Tutorial rapido jupyter notebook¶

Estrutura¶

Aplicativo jupyter notebook¶

O aplicativo jupyter notebook uma aplicao cliente-servidor que permite editar e rodar documentos notebook via um browser. Ele tambm possui um dashboard.

Notebook Dashboard¶

O dashboard de notebooks o componente que mostrado assim que abrimos o jupyter. Ele utilizado principalmente para a interao com o sistema de arquivos e controle sobre os documentos notebooks e o aplicativo jupyter notebook.

Documentos notebook¶

Os documentos notebook, so arquivos criados utilizando o aplicativo jupyter notebook. Eles podem conter tanto cdigo executavel, dependente de um "notebook kernel", como elementos de texto e grficos. Podemos dizer que eles so tanto documentos "human-readable" de uma anlise quanto um documento executavel que performa a anlise

Notebook kernel¶

O notebook kernel uma engine computacional que permite a execuo do cdigo contido no documento notebook. Para o curso utilizaremos uma engine python, porm poderiamos utilizar outras se fosse o caso.

Células¶

O notebook jupyter uma IDE baseada em clulas. Inicialmente temos 4 tipos, que podem ser incrementados com plugins:

Markdown¶

Clulas markdown so clulas de texto como essa. Utilizam formatao do tipo markdown, que uma extenso do html aceitando inclusive tags html como . Porm o markdown utilizado pelo notebook tambm foi extendido para aceitar matemtica com formatao tipo latex (ver formula abaixo). Finalmente dependendo das das extenses instaladas, podemos inserir resultados de cdigo python no texto.

  • Latex -> $x = y_c^{\frac{2}{7}}$
  • Execuo python dentro do markdown -> 4

Code¶

Clulas contendo cdigo a ser executado na linguagem escolhida no kernel. Para o caso deste curso todos os kernels sero de python, porm poderia ser utilizado para outras linguagens.

Heading¶

Deprecado, a forma correta de adicionar cabealhos agora pelo padrao do markdown

Raw nbconvert¶

Nbconvert uma facilidade do jupyter que permite converter o notebook para outros formatos, como por exemplo o html que usamos com as informaes de instalao. Essa clula nos permite colocar qualquer coisa que no deve ser convertida na hora dessa transformao. No muito utilizado

Essas clulas podem ser executadas em qualquer ordem (mesmo clulas de texto devem ser executadas para que sua formatao aparea), para isso voce pode usar o botao play no menu do topo, que equivalente ao atalho CTRL+Enter.

Exibição de gráficos¶

Usualmente a criao de grficos em python utiliza matplotlib. Para que esses grficos apaream no seu notebook, voc precisa utilizar a mgica %matplotlib. Existem dois tipos

  • %matplotlib inline: simplesmente joga o grfico gerado na saida do notebook.
  • %matplotlib notebook: gera um javascript que engloba o grfico e permite iterao com o grfico
In [15]:
import matplotlib.pyplot as plt
%matplotlib inline
plt.plot(vetor,vetor)
print("Grfico inline ")
